Git & GitHub
Team Workflow

A simple guide for working together without accidentally deleting the entire project 💀

REPOSITORY github.com/treesymse-source/Team
⚠️

STOP. READ THIS FIRST.

This project is shared between multiple people. Do not run random Git commands if you don't know what they do.

❌ Don't delete the whole project ❌ Don't use git push --force ❌ Don't use git reset --hard ❌ Don't delete files you don't understand
01

First time setup

Do this once when joining the project.

1

Install Git

Download and install Git. The default installation options are fine.

Download Git ↗
2

Clone the project

Open CMD in the folder where you want the project, then run:

git clone https://github.com/treesymse-source/Team.git
3

Enter the project

cd Team

Open the Team folder in VS Code.

02

Everyday workflow

Follow these steps every time you work on the project.

↓

1. Pull first

Get the newest version before starting work.

git pull
✎

2. Do your work

Edit only the files you need. Don't touch random files.

Example: You edit pages/info.html
?

3. Check changes

See what files you changed.

git status
+

4. Stage changes

Prepare your changes to be saved.

git add .
✓

5. Commit

Create a save point with a clear message.

git commit -m "Updated info page"
↑

6. Push

Upload your changes to GitHub.

git push
03

Quick reference

Commands you may need.

git statusSee what changed
git log --onelineSee previous save points
git remote -vCheck your GitHub connection
git revert COMMIT_IDSafely undo a previous commit
🚨

Something went wrong?

Don't panic. Don't delete the project.

  1. Stop and don't run more commands.
  2. Copy the complete error message.
  3. Tell the team owner what you were trying to do.
  4. Ask for help before using commands you don't understand.
Merge conflict? It usually means two people edited the same part of the same file. Do not delete the whole file. Ask for help.
⭐

The Golden Rule

PULL BEFORE WORKING.   →   Work   →   PUSH AFTER FINISHING.

If you are confused: STOP AND ASK. Do not delete the whole project 💀
